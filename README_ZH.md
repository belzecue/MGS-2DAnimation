# MGS-2DAnimation
- [English Manual](./README.md)

## 概述
- Unity制作2D动画插件包。

## 环境
- Unity 5.0 或更高版本。
- .Net Framework 3.5 或更高版本。

## 案例
- “MGS-2DAnimation\Scenes”目录下存有上述功能的演示案例，供读者参考。
